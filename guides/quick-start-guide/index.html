Quickstart guide

Prerequisites

  • Node ^12

  • YARN ^1.22 OR NPM ^6.10

  • PHP ^7.0.0

Get started using the automated command line tool. This command creates a Wingsuit demo project, which uses the atomic design guidelines to structure the patterns.

npx @wingsuit-designsystem/cli init

After the installation succeeds, Storybook will open and you can browse the Wingsuit demo page.

Drupal

To get started with Drupal and UI Patterns use the composer Wingsuit kickstarter.

composer create-project wingsuit-designsystem/wingsuit-kickstarter wingsuit-kickstarter --stability dev --no-interaction
cd wingsuit-kickstarter && ./vendor/bin/blt setup

The Kickstarter uses Tailwind. Adjust in blt/blt.yml

command: &#39;[ ! -d &quot;wingsuit&quot; ] &amp;&amp; npx @wingsuit-designsystem/cli init --skip-install; cd wingsuit &amp;&amp; yarn install&#39;

to:

command: &#39;[ ! -d &quot;wingsuit&quot; ] &amp;&amp; npx @wingsuit-designsystem/cli init -k bootstrap --skip-install; cd wingsuit &amp;&amp; yarn install&#39;

After that run:

cd wingsuit-kickstarter &amp;&amp; ./vendor/bin/blt setup

Bootstrap

With the -k option you can install other starter kits than Tailwind.

npx @wingsuit-designsystem/cli init -k bootstrap
